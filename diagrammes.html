Architecture du logiciel et diagrammes

Vue d'ensemble de l'application

L'application se compose de deux modes d’interaction :

  • Mode Console (CLI) : Destiné aux utilisateurs expérimentés, offrant une interface de ligne de commande pour effectuer des opérations sur les fichiers et leurs métadonnées.
  • Mode Graphique (GUI) : Conçu pour les utilisateurs moins expérimentés, permettant une gestion des images via une interface visuelle intuitive.
